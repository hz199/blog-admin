# vue-node

## Project setup
```
yarn install
```

### Compiles and hot-reloads for development
```
yarn run dev
```

### Compiles and minifies for production
```
yarn run build
```

### backEnd development
```
yarn run dev:server
```

### backEnd production
```
yarn run start
```

### Lints and fixes files
```
yarn run lint
```
